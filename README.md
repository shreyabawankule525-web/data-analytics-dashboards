# Data Analytics Dashboards

This repository contains Power BI dashboards created as part of my data analytics learning journey.

## Tools Used
- Power BI
- Microsoft Excel
- Data Visualization
- Data Analysis

## Projects

### 1. Pizza Sales Dashboard
This dashboard analyzes pizza sales data to identify key business insights.

Key Insights:
- Total revenue and order trends
- Best selling pizza categories
- Peak sales time analysis
- Customer purchasing behavior

### 2. Blinkit Sales Dashboard
This dashboard analyzes Blinkit sales data to understand product performance and category distribution.

Key Insights:
- Category wise sales performance
- Product distribution
- Sales trends and patterns
